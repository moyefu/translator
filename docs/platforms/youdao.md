## Application Steps

1. Visit [Youdao Zhiyun Open Platform](https://ai.youdao.com/)
2. Register and log in to your Youdao developer account
3. Click "Console" to enter the management interface
4. Create an application and select "Translation API" service
5. Complete verification (personal verification is sufficient)
6. Get Application ID and Application Key

## Notes

- Youdao Translate API requires two parameters: Application ID and Application Key
- Free quota: 500,000 characters per month for individual developers
- Beyond free quota: ¥49 per million characters
