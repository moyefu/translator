# 有道翻译 API 申请指南

## 申请步骤

1. 访问 [有道智云开放平台](https://ai.youdao.com/)
2. 注册并登录有道开发者账号
3. 点击「控制台」进入管理界面
4. 创建应用，选择「翻译 API」服务
5. 完成认证（个人认证即可）
6. 获取应用 ID 和 应用密钥

## 相关链接

## 注意事项

- 有道翻译 API 需要应用 ID 和 应用密钥 两个参数
- 免费额度：个人开发者每月 50 万字符
- 超出免费额度后按 49 元/百万字符计费
