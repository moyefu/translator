## 申请步骤

1. 访问 [腾讯云官网](https://cloud.tencent.com/)
2. 注册并登录腾讯云账号
3. 进入 [机器翻译服务](https://console.cloud.tencent.com/tmt)
4. 点击「立即开通」按钮
5. 完成实名认证
6. 创建 API 密钥，获取 SecretId 和 SecretKey

## 注意事项

- 腾讯翻译 API 需要 SecretId 和 SecretKey 两个参数
- 免费额度：新用户首月可获得 100 万字符的免费额度
- 超出免费额度后按 49 元/百万字符计费
