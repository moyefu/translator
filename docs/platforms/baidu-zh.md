# Baidu Translate API Application Guide

## Application Steps

1. Visit [Baidu Translation Open Platform](https://fanyi-api.baidu.com/)
2. Register and log in to your Baidu developer account
3. Click the "Activate Now" button
4. Choose an appropriate plan (there is a free quota)
5. Complete verification (personal verification allows 1 million characters/month for free)
6. Create an application and get App ID and Key

## Notes

- Baidu Translate API requires two parameters: App ID and Key
- Free quota: 1 million characters per month after personal verification
- Beyond free quota: ¥49 per million characters
