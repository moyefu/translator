# 谷歌翻译 API 申请指南

## 申请步骤

1. 访问 [Google Cloud 控制台](https://console.cloud.google.com/)
2. 注册并登录 Google Cloud 账号
3. 创建一个新的项目
4. 在左侧菜单中选择「API 和服务」>「库」
5. 搜索并启用「Cloud Translation API」
6. 在左侧菜单中选择「API 和服务」>「凭据」
7. 创建一个新的 API 密钥
8. 配置 API 密钥的使用限制（可选）

## 相关链接

- [Google Cloud 控制台](https://console.cloud.google.com/)
- [Cloud Translation API 文档](https://cloud.google.com/translate/docs)
- [Cloud Translation API 价格](https://cloud.google.com/translate/pricing)

## 注意事项

- 谷歌翻译 API 只需要 API 密钥一个参数
- 免费额度：新用户每月 50 万字符免费，且新用户赠 300 美元（90 天内有效）
- 超出免费额度后按 $20/百万字符计费
